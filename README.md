## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn or pnpm

### Installation

1. Clone the repository
```bash
git clone git@github.com:ups216/copilot-website.git
cd copilot-website
```

2. Install dependencies
```bash
npm install
# or
yarn
# or
pnpm install
```

3. Start the development server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

4. Build for production
```bash
npm run build
# or
yarn build
# or 
pnpm build
```
